# Privacy Policy

Last Updated: March 2026

## Overview

Wine Buddy is a web application that respects user privacy. We collect minimal data and do not collect personally identifiable information (PII).

## Data Collection

### What We Collect

We use **Sentry** for application performance monitoring. This includes:
- Response times for each endpoint
- Error logs and stack traces
- Basic device information (OS and browser type)
- Request paths and HTTP status codes

### What We Do NOT Collect

- **IP addresses** — Disabled for GDPR compliance
- **User identifiers** — No user accounts or tracking cookies
- **Request bodies** — Automatically filtered to prevent data leakage
- **Cookies** — Not tracked or logged
- **Personal information** — No name, email, or individual identifiers

## Why We Collect This Data

Performance monitoring data helps us:
- Identify and fix errors affecting the application
- Monitor endpoint response times
- Ensure reliable service across different devices and browsers
- Improve overall application stability and performance

## Data Security & Retention

- All data is transmitted over HTTPS (encrypted)
- Sentry compliance: [Sentry's Data Processing Agreement](https://sentry.io/privacy/)
- Data retention: Determined by your Sentry project settings (typically 90 days)

## Your Rights

Since we do not collect personally identifiable information, GDPR requires no consent banner or data subject rights requests. However, if you have questions about our data practices, please reach out.

## Contact

For privacy concerns, contact the project maintainer via GitHub Issues.
